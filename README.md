# LRMusicVolumeSlider (app code only)
 This is an small C# Winforms based GUI app that allows you to control the volume levels of the music in LIGHTNING RETURNS FINAL FANTASY XIII steam version. 
 <br>
 <br>
This repo only has the project code for the App, the batch scripts and only some of my patching tools. the ff13tool and the WPDPack tools that are both required for patching the volume of the game is not included as I am not the author of those tools.
<br>
<br>
You can download my compiled builds of this app from my mod page on nexus:
<br>
https://www.nexusmods.com/lightningreturnsfinalfantasy13/mods/8
<br>

The compiled build from nexus will include the ff13tool and the WPDPack tool as well. so if you want to use it along with your compiled app with this code, then please contact the respective authors behind those tools for permission if you want to use it in your own compiled app.
<br>

The ff13tool that this app uses is also made by the same author as the WPDPack tool. its a modified version of the original ff13tool by FluffQuack but has some changes in it to make it somewhat different from the original.
